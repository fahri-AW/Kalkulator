Contoh Kode Python untuk Penyelesaian Integral

Kode ini menggunakan library SymPy (Python) untuk menyelesaikan integral secara simbolik.

**Catatan:** Kode ini adalah teks dan memerlukan instalasi Python beserta library SymPy (pip install sympy) untuk dapat dieksekusi.

1. Integral dengan Metode Substitusi

from sympy import integrate, Symbol

# Definisikan variabel
x = Symbol('x')

# Contoh soal: Integral dari 2x * (x^2 + 1)^3 dx
# Asumsi substitusi: u = x^2 + 1, du = 2x dx
ekspresi_substitusi = 2 * x * (x**2 + 1)**3

# Hitung integral tak tentu
hasil_substitusi = integrate(ekspresi_substitusi, x)

print("### Integral dengan Substitusi ###")
print(f"Soal: Int(2x * (x^2 + 1)^3) dx")
print(f"Hasil: {hasil_substitusi} + C")

# Hasilnya adalah: (x**2 + 1)**4/4 + C

2. Integral dengan Metode Parsial

from sympy import integrate, Symbol

# Definisikan variabel
x = Symbol('x')

# Contoh soal: Integral dari x * cos(x) dx
# Asumsi parsial: u = x, dv = cos(x) dx
ekspresi_parsial = x * Symbol('cos')(x)
hasil_parsial = integrate(ekspresi_parsial, x)

print("\n---")
print("### Integral dengan Parsial ###")
print(f"Soal: Int(x * cos(x)) dx")
print(f"Hasil: {hasil_parsial} + C")

# Hasilnya adalah: x*sin(x) + cos(x) + C
